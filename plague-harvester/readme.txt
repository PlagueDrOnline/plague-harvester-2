Plague Harvester lets site owners crawl public content, import it into WordPress, export imported posts, and manage affiliate and banner campaigns from a single admin panel.

== Description ==

Plague Harvester provides:

* URL crawling and content extraction
* Draft import of crawled posts
* JSON / CSV export of imported posts
* Banner management for affiliate or ad campaigns
* Affiliate settings dashboard and metrics panel

== Installation ==

1. Upload the `plague-harvester` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Open the new 'Plague Harvester' menu item in the admin sidebar.
4. Enter a URL to crawl, then import the generated content.

== Usage ==

* Use the Crawl tab to fetch page content from a URL.
* Use the Import tab to review and import crawled content into WordPress as a draft.
* Use the Export tab to download imported posts as JSON or CSV.
* Use the Banners tab to save banner links and images for campaigns.
* Use the Affiliate tab to store affiliate code and partner URLs.
